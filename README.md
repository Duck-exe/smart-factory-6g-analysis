# Manufacturing Process Health and Operational Efficiency Analysis in 6G-Enabled Smart Factories

This project analyzes machine health, production efficiency, quality defects, error rates, and 6G network performance in a smart factory environment.

## Features
- Factory health overview
- Machine health index calculation
- Production and quality diagnostics
- 6G network latency and packet loss analysis
- Efficiency status breakdown
- Interactive Streamlit dashboard

## Technologies Used
- Python
- Pandas
- NumPy
- Plotly
- Streamlit

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
